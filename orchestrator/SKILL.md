# 🪄 Prime Orchestrator

## Overview

The Prime Skill Set is your high-performance operating system. This orchestrator helps you identify the right module for your current engineering phase.

## Discovery Map

```
What are you doing?
    │
    ├── Thinking/Refining? ──→ spec-driven-dev
    ├── Planning tasks? ─────→ task-breakdown
    ├── Implementing code? ──→ portfolio-craft
    │   ├── Growth/SEO? ─────→ growth-engine
    │   └── Security/Infra? ─→ sys-architect
    ├── Verifying work? ─────→ verification-lab
    └── Need checklists? ────→ checklists-library
```

## Core Behaviors

1. **Surface Assumptions**: "I'm assuming X... correct me or I proceed."
2. **Push Back**: Don't be sycophantic. If a design violates performance or security, say so.
3. **Evidence First**: Never say "it works" without a test result or screenshot.
4. **No AI Default**: Every UI element must have a brand-conscious rationale.

## Meta-Lệnh (Commands)

- **/spec**: Activate `spec-driven-dev`.
- **/plan**: Activate `task-breakdown`.
- **/test**: Activate `verification-lab`.
- **/craft**: Activate `portfolio-craft`.
- **/growth**: Activate `growth-engine`.
- **/sys**: Activate `sys-architect`.
